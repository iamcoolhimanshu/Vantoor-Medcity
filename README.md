# 📖 About

**Vantoor Medcity** is a modern Hospital Management System designed to simplify hospital operations through automation, AI-powered tools, analytics, appointment management, and workflow automation.

The application provides a clean, responsive, and scalable platform for managing healthcare services efficiently.

# 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/iamcoolhimanshu/vantoor-medcity.git
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```
